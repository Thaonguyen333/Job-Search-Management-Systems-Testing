# Job-Search-Management-Systems-Testing
## Project Overview

This project contains manual test cases designed for a Job Search System based on the Software Requirements Specification (SRS).

## Scope

The following modules were covered:

- User Registration
- Login
- Job Search
- Apply Job
- Employer Job Posting
- Edit Job
- Delete Job
- Candidate List
## Test Design Techniques

- Functional Testing
- Boundary Value Analysis
- Negative Testing
- Input Validation
- Positive Testing

## Deliverables

- Test Case Document (Excel)

## Tools

- Microsoft Excel
- SQL Server
